# 一. 数组
- 1. 创建数组
-   利用数组 new
-      var 数组名=new Array(); 
-      var arr=new Array; 用来创建一个新的空数组
-    利用数组字面量创建数组[]
-       var arr = []
- 注意：数组里面的数据要用，分隔
- 2. 访问数组元素
- 方法：数组名[索引号]     （索引号从0开始）
- 3. 遍历
- 将数组中的每个元素从头到尾都访问一次
- arr.length  动态监测数组元素个数
- 4. 新增数组元素
- 方法：修改索引号，追加数组元素
-         eg. var arr=[1, 2, 3]
-             arr[3]=4
- 注意：如果arr[]里面已经有了值，则会修改元素
       不要直接给数组名赋值，否则会覆盖掉以前的数据
