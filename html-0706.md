## 二. 表单标签

- 表单域:`<form>`标签
- 传递范围内表单元素:`<form>`
`<form action="url地址" method="提交方式" name="表单域名称">各种表单元素控件`  `</form>`

- 表单元素:`<input>`单标签
 `<input type="属性">`

`<label>` : 在浏览器里面自动聚焦
 `<label for="属性">`

`<select>` : 下拉列表(至少有两组选项)
`<option>` 中定义selected="selected"时为默认选项·

```
    <select>
         <option>xxx</option>
          <option>xxx</option>
     </select>
```

- 多行输入文本框:`<textarea>`
- cols: 每行字数   rows: 显示的行数
```
   <textarea rows="x" cols="x">

     文本内容

   </textarea>
```
